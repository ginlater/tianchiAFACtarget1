# MIX线优化日志（分域最优拼装）

## 定位
每域从其最强运行取件(答案+该域真实逐题账),行行自洽可审计。总分公式的全局优化器。

## 方法
全局求解器: 对全部运行×5域计算 价值=0.5×键_d − 4×(百万token_d), 逐域argmax。
合规红线: 排除人工审计过的答案目录(b_v4/b_probe_fix)——那是"未申报改写"。
