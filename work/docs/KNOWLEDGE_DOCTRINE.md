# 知识结构定律（三矿合成, 7-25凌晨定稿）

## 总定律
**慢性错题的病理几乎是一元的：信息不在上下文（漏检系 7/11=64%）。
一切真正的治愈都来自新的知识结构把缺失信息确定性送进上下文；
判断层算力（投票/复核/砸钱）治愈记录为零。**

## 五条设计原理(MSIS框架, 全部有账可查)
1. **补维不加量**: 诊断缺哪类信息(数值绑定/身份消歧/覆盖保障/表达对齐), 用最小结构恰好补那一维
   (fin_facts2零token治fin_b_012 vs full10烧4.59M仍90键)
2. **压缩形态必须匹配域材质**: 闭合schema→单元格表; 叙述→卡片; 开放条款→原句+全文卡
   (同为速查表fin胜ins败; 同为卡片reg20/20、E1fin7/20)
3. **有损结构不得垄断信道**: 压缩件只能叠加+原文回退(finD替代式12/20 vs 补丁式18)
4. **验证的价值=错误分布的独立性**: 复核只花在与主答错误不相关的信道
   (跨代异构+5键零增账; r2条件化双域证伪; r1×3花1.26M买0键)
5. **摊薄仅限共享成本**: 逐题边际MSIS一字不能摊(finsB 4.5k破24k之题 vs E1摊没表格7/20)

## 发现方法论(三次突破的共同起点)
